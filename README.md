# COVID-GENE-EXPRESSION
Final Project for genomics Data Analysis. I am comparing gene expression between COVID-19 and healthy individuals.
GSE152418_p20047_Study1_RawCounts file is the raw data used for the analysis
manuscript_ambrosio.RMD is the file used to generate the manuscript, with citations
final_project_ambrosio.RMD is the analysis file with all the code and results included, and informal results.
references.bib is the texxt file used to generate in text citations for the RMD manuscript file
